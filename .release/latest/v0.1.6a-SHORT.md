# Allium v0.1.6a

**Wiki:** https://github.com/castledking/Allium/wiki

## Highlights

- **Chat Channels** — `/local` and `/l` for radius chat, `/global` and `/g` for global. DiscordSRV integration with per-channel relay control.
- **ModGuard** — client-mod detection framework with brand/channel/FML probes, evidence logs, alerts, and kick actions. Configurable blacklist/whitelist.
- **Animated Gradient Names** — `%gradientdisplayname%` outputs animated MiniMessage gradients for players with `allium.gradientname`.
- **Staff Locator Bar** — `/locatorbar` staff mode shows all visible players regardless of party radius.
- **Item Renamer** — new single-use item granting temporary `/rename` access.
- **Safer End Gateways** — gateway teleports near the main island search for safe same-level landing positions.
- **Fixes** — nickname persistence with TAB, `/local`/`/global` alias routing, DiscordSRV console mirroring, raw TAB placeholder leak, far-away animated name stutter, same-name nickname rejection, DB pool memory reduction.
- **Internal** — package rename to `codes.castled.allium`, release automation via GitHub Actions.
